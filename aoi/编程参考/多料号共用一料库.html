多料号共用一料库

多料号共用一料库

导入单面CAD

Qubit AOI支持同外观不同料号的料件共用一个检测库，并可实现库自动连接。同外观元件共用一个检测库可以减少元件库数，提高检测效率，同时方便管理。

例如，常见的0603电容外观完全一致，因电性能差异，料号不同。

如果仅需要对电路板进行单面检测，则只需要导入单面CAD数据。

操作步骤如下：

  1. 点击 [ 切换面 ] 切换到正确的检测面。
  2. 依次点击菜单项 [ AOI ] → [ 导入 ] → [ 导入单面CAD ] 后弹出CAD文件选择对话框。
  3. 选择需要导入的CAD文件，点击 [ 打开 ] 。
  4. 系统自动识别 位号、料号、封装、X、Y、角度 后，在左边的元件组列表就能看到导入的元件信息了。
