制作BGA检测库

制作BGA检测库

BGA的常规检测通常可以使用以下算法：

  • 窗口1：使用最大值算法，用以定位元件，检测是否缺件。
    • 窗口1.1：使用相对偏移算法，用以检测元件偏移。
  • 窗口2：使用相对偏移算法，用以检测元件偏移。

操作步骤如下：

  1. 创建窗口1，使用最大值算法（单窗口）检测是否缺件：

    1. 在左侧元件组列表找到元件类型为BGA的元件组， 双击该组， 可以看到BGA元件列表，双击选择需要编程的BGA元件。
    2. 设定元件参数：元件长度 元件宽度。
    3. 切换到 [ 检测窗口 ] 面板，选择 [ 单窗口 ] ， 此时可以在右侧检测窗口列表中看到一个新的检测窗口，点击 [ 检测算法 ] 下拉菜单， 在检测算法列表选择“平均值”。
    4. 设置检测窗口的宽度 高度，调整其大小为调略大于元件本体尺寸。
    5. 依次点击菜单项 [ IPE ] → [ RGBA颜色抽取 ] ，或者 [ HSVA颜色抽取 ] ，进入颜色抽取界面，抽取电路板底板颜色，使电路板底板和元件本体明显区分，元件本体为亮，具体操作方法请参见：RGBA颜色抽取、HSVA颜色抽取。
    6. 设置算法的区域高度 区域宽度为元件本体尺寸； 设置下限= 0.6。
    7. 设置检测窗口的报警类型为“缺件”。

  2. 创建窗口1.1，使用相对偏移算法（单窗口）检测元件偏移:

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] ，点击 [ 降级 ] 。
    2. 点击 [ 检测算法 ] ， 在检测算法列表选择“相对偏移”，根据元件工艺规格设置最大X偏移 最大Y偏移。
    3. 设置检测窗口的报警类型为“偏移”。

  3. 创建窗口2，使用相对偏移算法（单窗口）检测元件偏移:

    1. 创建顶级窗口： [ 单窗口 ] 。
    2. 点击 [ 检测算法 ] ， 在检测算法列表选择“相对偏移”。
    3. 选择与窗口1相同IPE。
    4. 设置检测窗口的宽度 高度至合适大小。
    5. 设置检测窗口的X Y,移动窗口至BGA外，紧贴BGA本体。
    6. 根据元件工艺规格设置最大X偏移 最大Y偏移。
    7. 设置检测窗口的报警类型为“偏移”。

  4. 重复制作多个类似窗口2的窗口，环绕BGA本体。
