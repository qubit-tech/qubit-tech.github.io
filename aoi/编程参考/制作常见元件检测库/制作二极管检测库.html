制作二极管检测库

制作二极管检测库

二极管的常规检测通常可以使用以下算法：

  • 窗口1：使用平均值算法，用以检测是否缺件。
  • 窗口2：使用模板匹配算法，用以定位元件的位置。
    • 窗口2.1：使用相对偏移算法，用以检测元件偏移。
    • 窗口2.2：使用2端角度侦测算法，用以检测元件角度a偏转。
    • 窗口2.3：使用智能侦测算法，用以检测两边的焊锡。
    • 窗口2.4：使用亮度抽取算法，用以检测两边的焊锡。
    • 窗口2.5：使用亮度抽取算法，用以检测两边的焊锡。
    • 窗口2.6：使用模板匹配算法，用以定位元件上字符 的位置。
      • 窗口2.6.1：使用字符验证算法，用以检查字符。
      • 窗口2.6.2：使用极性检测算法，用以检测是否极反。

操作步骤如下：

  1. 创建窗口1，使用平均值算法（单窗口）检测是否缺件：

    1. 在左侧元件组列表找到元件类型为二极管的元件组， 双击该组， 可以看到二极管元件列表，双击选择需要编程的二极管元件。

    2. 设定元件参数：元件长度 元件宽度。

    3. 切换到 [ 检测窗口 ] 面板，选择 [ 单窗口 ] ， 此时可以在右侧检测窗口列表中看到一个新的检测窗口，点击 [ 检测算法 ] 下拉菜单， 在检测算法列表选择“平均值”。

    4. 设置检测窗口的宽度 高度，调整算法的检测范围为元件中心一小块区域，不包含焊盘。

    5. 依次点击菜单项 [ IPE ] → [ RGBA颜色抽取 ] ，或者 [ HSVA颜色抽取 ] ，进入颜色抽取界面，抽取电路板底板颜色，使电路板底板和元件本体明显区分，底板为亮，元件本体为暗，具体操作方法请参见：RGBA颜色抽取、HSVA颜色抽取。

    6. 设置算法的上限 = 0.3 下限= 0。

    7. 设置检测窗口的报警类型为“缺件”。

  2. 创建窗口2，使用模板匹配算法（单窗口）定位元件位置：

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] 。
    2. 点击 [ 检测算法 ] 下拉菜单， 在检测算法列表选择“模板匹配”。
    3. 选择合适的IPE可以减少服务器运算负担，一般建议选择r。
    4. 点击 [ 取像 ] 进入模板取像模式，蓝色方框表示取像范围，可以用鼠标左键拖曳调整取像范围至切合目标元件图像（包括焊盘），点击 [ 取像 ] ， 蓝色方框内的图像将被取为模板。
    5. 设置算法的下限= 0.6。
    6. 设置检测窗口的宽度 高度，调整算法的检测范围为元件可能出现的范围；设置报警类型为“缺件”。

  3. 创建窗口2.1，使用相对偏移算法（单窗口）检测元件偏移:

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] ，点击 [ 降级 ] 。
    2. 点击 [ 检测算法 ] ， 在检测算法列表选择“相对偏移”，根据元件工艺规格设置最大X偏移 f最大Y偏移。
    3. 设置检测窗口的报警类型为“偏移”。

  4. 创建窗口2.2，使用2端角度侦测测算法（单窗口）检测元件角度偏转:

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] 。
    2. 点击 [ 检测算法 ] ， 在检测算法列表选择“2端角度侦测”。
    3. 选择合适的IPE来凸显两端焊盘，一般建议选择r。
    4. 设置检测窗口的报警类型为“偏移”。

  5. 创建窗口2.3，使用智能侦测算法（双窗口）检测元件两边的焊锡:

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 双窗口 ] 。
    2. 设置检测窗口的角度，对窗口进行旋转；设置检测窗口的半径，调整2个窗口之间的距离，使得其内侧紧靠元件电极；设置检测窗口的宽度 高度，使窗口大小与爬锡处相仿。
    3. 点击 [ 检测算法 ] ， 在检测算法列表选择“智能侦测”，调整算法的区域宽度 区域高度，一般都设置为0.2；点击 [ 更新 ] 模板；最后根据工艺设置上限 下限，一般设置在0.1~0.2之间。
    4. 设置检测窗口的报警类型为“虚焊”。
    5. 对算法进行训练和批量训练。

  6. 创建窗口2.4，使用亮度抽取算法（双窗口）检测检测元件两边的焊锡:

    1. 选中前面创建的智能侦测双窗口，点击 [ 克隆窗口 ] ，此时会在同一级别克隆出一个同样的双窗口，将检测算法修改为“亮度抽取”。
    2. 选择合适的IPE抽取红色，一般选择2*r-g-b。（由于如果爬锡良好，顶部的红色光将大部分被反射，进入进入相机镜头红色光应该很少，因此抽取红色可以判断爬锡情况）
    3. 设置算法参数：阈值一般在0.2~0.3之间；上限一般设置为0.3；下限为0。
    4. 设置检测窗口的报警类型为“少锡”。

  7. 创建窗口2.5，使用亮度抽取算法（双窗口）检测检测元件两边的焊锡:

    1. 选中前面创建的亮度抽取双窗口，点击 [ 克隆窗口 ] ，此时会在同一级别克隆出一个同样的双窗口，将检测算法修改为“亮度抽取”。
    2. 选择合适的IPE抽取蓝色，一般选择b。（通过检验蓝色光反射程度判定上锡处锡面斜率。如果斜率足够（上锡良好）镜头可以获取较多蓝色光；如果锡面平滑（缺锡），镜头无法获取足够蓝光。）
    3. 设置算法参数：阈值一般在0.07~0.1之间；上限一般设置为1；下限为0.6。
    4. 设置检测窗口的报警类型为“漏铜”。

  8. 创建窗口2.6，使用模板匹配算法（单窗口）用以定位二极管上的字符。

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] 。
    2. 点击 [ 检测算法 ] 下拉菜单， 在检测算法列表选择“模板匹配”。
    3. 依次点击菜单项 [ IPE ] → [ RGBA颜色抽取 ] ，进入颜色抽取界面，将字符与背景区分开，一般将字符与背景明显区分开，一般字符为亮为亮，背景为暗，具体操作方法请参见：RGBA颜色抽取。
    4. 点击 [ 取像 ] 进入模板取像模式，蓝色方框表示取像范围，可以用鼠标左键拖曳调整取像范围至刚好切合目标元件上的字符，点击 [ 取像 ] ， 蓝色方框内的图像将被取为模板。
    5. 设置检测窗口的宽度 高度，为仅包含字符的图像大小，同时由于字符方向有可能不同，需要将模式修改为“双向”。
    6. 设置算法的下限= 0.2。
    7. 设置检测窗口的报警类型为“错件”。
    8. 重复4)中的取像步骤，并点击 [ 更新 ] ， 为了降低误报，一般需收集10个左右样板。

  9. 创建窗口2.6.1，使用字符验证算法（单窗口）用以检查字符。

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] ，点击 [ 降级 ] 。
    2. 点击 [ 检测算法 ] 下拉菜单， 在检测算法列表选择“字符验证”。
    3. 设置检测窗口的宽度 高度，为仅包含字符的图像大小。
    4. 设置算法的区域宽度 区域高度，一般设置为0.1~0.2之间。
    5. 设置算法的下限，一般设置为 0.2。
    6. 设置检测窗口的报警类型为“错件”。
    7. 对算法进行训练和批量训练。

  10. 创建窗口2.6.2，使用极性检测算法（单窗口）检测是否极反:

    1. 在 [ 检测窗口 ] 面板，选择创建 [ 单窗口 ] 。
    2. 点击 [ 检测算法 ] ， 在检测算法列表选择“极性检测”。
    3. 设置检测窗口的报警类型为“极反”。
