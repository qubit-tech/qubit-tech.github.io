制作mark检测库

制作MARK检测库

MARK检测库制作是检测程序制作过程中最重要的环节，只有找到MARK，才能校正图像和元件位置，后续各类元件检测库制作都将依赖于MARK的确定。 MARK通常分为主MARK和子MARK（拼板MARK），主MARK主要用于校正整板图像，而子MARK主要用于校正拼板中元件的位置。

对于一个检测程序来说，至少需要1个主MARK；当然主MARK越多，图像校正越准确，通常建议设置电路板4个角位置的4个主MARK；或者设置电路板对角位置的2个主MARK。而拼板MARK，也就是子MARK，每块拼板上建议设置1个。

操作步骤如下：

  1. 在左侧元件组列表找到元件类型为MARK的元件组， 双击该组， 可以看到MARK元件列表，双击选择需要编程的MARK元件。

  2. 设定元件参数：元件长度 元件宽度 拼板，请注意：请设置主MARK元件的拼板为0，子MARK元件的拼板与其所在拼板一致。

  3. 点击菜单项 [ 检测库 ] → [ 新建库 ] ，此时自动切换到 [ 检测窗口 ] 面板，选择 [ 单窗口 ] ，此时在右侧检测窗口列表中可以看到一个新的检测窗口，点击 [ 检测算法 ] 下拉菜单， 选择带有定位功能的算法定位MARK，通常可以使用的算法有“模版匹配”、“圆侦测”、“最大值”、“最小值”等等。 本例中在检测算法列表选择“模版匹配”。在实际使用时，根据图像的质量情况，可以组合使用多种算法以精确定位MARK。

  4. 点击 [ 取像 ] 进入模版取像模式，蓝色方框表示取像范围，可以用鼠标左键拖曳调整取像范围至刚好切合MARK图像，点击 [ 取像 ] ， 蓝色方框内的图像将被取为模版。

  5. 最后可以设置检测窗口的宽度、高度，调整检测算法的检测范围。

  6. MARK检测库制作完成后，请点击菜单项 [ 主页 ] → [ 应用MARK ] 。如果MARK检测库设置正确，应用MARK之后，将看到CAD中导入的元件位置与图像中的元件位置吻合，这表示检测程序已经找到MARK，从而校正了图像和元件位置。请注意，后续各类元件检测库制作都必须在菜单项 [ 应用MARK ] 被选中的状态下进行。
