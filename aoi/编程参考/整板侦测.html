如何使用整板侦测

整板侦测

整板侦测功能可以用于辅助检测电路板任意位置（包含无CAD坐标）的异常，常用于检测多件、锡珠、刮痕等不良。

检测原理

  1. 使用多块电路板的完整图像进行学习训练，建立图像的标准模型。
  2. 使用训练得到的标准模型检测待测试图像上的异常点。
  3. 根据工程师设置的参数，选出最有可能的异常点并报警。

使用步骤

  1. 按正常流程制作完标准的检测程序。
  2. 点击 [ 主页 ] → [ 整板侦测 ] ，进入整板侦测调试界面。
  3. 勾选启用侦测和启用训练度后，点击 [ 关闭 ] 退出整板侦测调试界面。
  4. 保存程序后，进入运行检测模式。这时已经进入训练学习模式，每次检测后会自动训练学习当前的图像，并增加样本计数。当样本计数达到20以上时，就会自动使用训练学习得到的标准模型检测当前的图像。
  5. 可以使用 [ 保存数据 ] 保存标准模型到存储器中，使用 [ 加载数据 ] 从存储器中加载已保存的标准模型。
  6. 可以使用整板侦测调试界面里提供的调试功能微调检测参数，并配合使用整板侦测(+)整板侦测(-)，提高检出率并降低误报率。

参数设定

参数 描述
侦测模式 指定使用整板侦测的检测范围：
1. 全板范围: 表示在全板范围内启用检测。
2. 窗口范围: 在使用算法整板侦测(+)的检测窗口范围内启用检测。
启用侦测 指定在运行检测时启用整板侦测功能。
启用训练 指定在运行检测时启用整板侦测训练功能。
阈值 指定检测数据匹配的合格下限值，小于这个参数值时表示异常点。
报警尺寸 指定报警点的最小面积（单位：平方毫米），可以用这个参数过滤灰尘等干扰，减少误报。
1. 异常点面积小于指定值时不报警。
2. 异常点面积大于等于指定值时报警。
留边(左) 留边(右) 留边(上) 留边(下) 指定电路板边不检测区域的宽度（单位：毫米），可以用于过滤板边毛刺等干扰，减少误报。

注意事项

  • 整板侦测对图像的稳定性要求比较高，因此需要做好全局的MARK点检测。这样能更好的校正图像，提高标准模型的精准度，从而提高检出率并降低误报率。一般建议制作PCB4个角落的4个以上MARK。
  • 整板侦测需要20以上的样本才能开始启用。
  • 不要使用同一块PCB或同一张图像进行反复训练，这样可能会使标准模型仅适合这块PCB板，从而得不到通用的标准模型。
  • 虽然整板侦测对于样本没有要求，但是在样本数较少时，最好都使用合格板的图像来训练学习，这样能更快的训练出标准模型。
