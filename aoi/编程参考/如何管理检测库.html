如何管理检测库

管理检测库

Qubit AOI提供完备的检测库管理功能，能够大幅度提高编程效率，防范首件错件，从而保证检出率、降低误报率。

导出检测库

[ 导出检测库 ] 功能将选中检测库或当前程式所有检测库导出保存在指定文件夹内。

操作步骤如下：

  1. 导出当前选中的检测库：点击菜单项 [ 检测库 ] → [ 链接库 ] → [ 导出检测库 ] 。
  2. 导出所有检测库：点击菜单项 [ AOI ] → [ 导出 ] → [ 导出检测库 ] 。

导入检测库

[ 导入检测库 ] 功能将已经制作好的检测库文件导入当前程序，并根据对应关系自动链接。

操作步骤如下：

  1. 导入指定的检测库文件至当前选中的元件组：点击菜单项 [ 检测库 ] → [ 链接库 ] → [ 导入检测库 ] 。
  2. 在当前程序中导入多个检测库并自动链接：点击菜单项 [ AOI ] → [ 导入 ] → [ 导入检测库 ] ，选择需要导入的检测库文件，可多选。

导入关联检测库

[ 导入关联检测库 ] 功能将选中的检测库中与当前程序中存在对应关系的部分导入，并根据对应关系自动链接。“对应关系”包括检测库名称与料号名相同、检测库名称与组名相同，如果编程人员曾经在本机使用过 [ 保存组信息 ] 功能来保存检测库与组的对应关系，那么在导入检测库时，与当前程序中的组名存在对应关系的检测库也会被导入，并自动链接到组。

操作步骤如下：

  1. 在当前程序中导入多个检测库并自动链接：点击菜单项 [ AOI ] → [ 导入 ] → [ 导入关联检测库 ] ，选择需要导入的检测库文件，可多选。

清理库

[ 检测库 ] 功能将清理掉程序内未链接使用的检测库，减少程式负担。在使用“导入检测库“，完成程式制作后建议使用该功能。

操作步骤： 点击菜单项 [ 检测库 ] → [ 清理库 ] 。

标准检测库

“清理库”功能用于制作普通检测库模板。

操作步骤如下：

  1. 完成检测库制作，使用 [ 导出检测库 ] 功能导出。
  2. 将导出的检测库存放在目录C:\aoi\lib\std下。
  3. 点击菜单项 [ 检测库 ] → [ 新建库 ] → [ 标准检测库 ] ，选择需要使用的标准库，其作用相当于新建库（库名称自动使用组名称），并将标准检测库窗口克隆至该新建库内。
