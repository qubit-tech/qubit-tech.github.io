编辑CAD

编辑CAD

导入CAD后，点击菜单项 [ 元件 ] → [ 编辑CAD ] ，进入“编辑CAD”界面，通过编辑CAD可以调节元件坐标、方向与PCB一致。

功能介绍

  1. 在 [ 目标拼板 ] 下拉列表中指定待编辑拼板：
参数 描述
* 表示选中所有拼板，包括各子基板与主板（拼板号0）。
# 表示选中所有子基板，不包括主板（拼板号0）。
数字 表示选中对应数字的拼板。

  1. 在 [ 旋转角度 ] 输入框中，对选中的目标拼板进行旋转。

  2. 在 [ 左右偏移 ] 、 [ 上下偏移 ] 输入框中，对选中的目标拼板进行平移。

  3. 在 [ 复制行数 ] 、 [ 复制列数 ] 输入框中，对选中的目标拼板进行复制。

  4. 在 [ 行距 ] 、 [ 列距 ] 输入框中，调整相邻拼板的间距。

  5. 使用 [ 左对齐 ] 、 [ 上对齐 ] 实现目标拼板左对齐、上对齐；使用其下拉选项，可以实现右对齐、下对齐、居中。

  6. 使用 [ 左右翻转 ] 、 [ 上下翻转 ] 实现目标拼板翻转。

  7. 使用 [ 阵列扩展 ] 自动搜索与目标拼板相同（方向也需相同）拼板，并对此扩展，自动完成拼板复制。。

  8. 使用 [ 镜像定位 ] 自动搜索选中拼板附近，与其反180度拼板位置，并自动对正。（请注意，此功能并没有复制功能）

  9. 使用 [ 删除拼板 ] 删除选中拼板，与上述各功能需按下 [ 应用 ] 键方可生效不同， [ 删除拼板 ] 立即生效。

  10. 使用 [ 克隆拼板 ] 复制一份出选中拼板，位置完全一样。无需 [ 应用 ] ，即时生效 。

  11. 编辑CAD内，众多功能，除 [ 删除拼板 ] 、 [ 克隆拼板 ] 外，需按下 [ 应用 ] 方可生效。
