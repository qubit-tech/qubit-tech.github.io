智能判定

智能判定

算法说明

该算法通过分别采集各种合格和不良的图像样本，通过训练建立智能判定模型，计算待测图像的相似度。

参数设定

参数 描述
合格样本 采集的合格样本数。
不良样本 采集的不良样本数。
训练模型 表示当前训练模型的状态：空白表示判定模型未建立；“√”表示判定模型已建立；“√!”表示样本已变化，模型可能需要更新。
阈值 待测图像的相似度得分低于设定阈值为：合格；小于等于阈值为：报警。

采集样本

如果当前选中的检测窗口使用的是“智能判定”算法时，可以在左侧元件列表中选择元件添加为样本，操作步骤方式如下：

  1. 选中单个或者多个元件（按住Ctrl键或者Shift键，使用鼠标左键点击选择多个元件）。
  2. 在选中的元件上单击鼠标右键，弹出菜单，可以在菜单中选择 [ 添加为合格样本 ] 或者 [ 添加为不良样本 ] ，所选中的元件将被添加到样本库中。

训练对话框

样本采集完成后，点击训练模型右边的训练图标，可以打开训练对话框。训练对话框提供样本管理和模型管理功能。

样本管理

  • 样本将自动分页显示，每64个样本为一页，通过点击样本页下方的图标，可以进行翻页操作。
  • 通过鼠标左键点击选择样本，按住Ctrl键或者Shift键，可以进行多选操作。
  • 在选中的样本上单击鼠标右键，弹出菜单，可以在菜单中选择 [ 添加为合格样本 ] 或者 [ 添加为不良样本 ] 移动样本；可以选择 [ 删除 ] 从样本库中移除当前选中样本。

模型管理

  • 训练模型

    点击 [ 训练模型 ] 按钮，将使用当前的样本库训练智能判定模型。为了得到优化的模型，需要分别采样不同类型的合格样本与不良样本，建议每种类型的合格和不良样本数大于等于3个；在不良样本数较少的情况下，合格样本数尽量接近不良样本数。

    是否选择合适的图像对称性，将直接影响训练结果，具体选项请参考下表：

图像对称性 描述
不对称 图像不对称。
X轴对称 图像上下对称。
Y轴对称 图像左右对称。
X轴和Y轴对称 图像上下左右对称，例如：贴片电容元件 。
旋转对称 图像旋转180°对称。
中心对称 图像中心对称，例如：波峰焊点、圆形MARK点。

  • 测试模型

    模型训练完成后，点击 [ 测试模型 ] 按钮，可以使用生成的模型对样本库中的样本进行测试，测试不通过的样本会被标记出来，请检查样本是否合适。

  • 清除模型

    如果发现当前模型不合适，需要优化样本，可以点击 [ 清除模型 ] 按钮，清除当前模型。

注意事项

  • 该算法只能使用3个颜色通道的IPE，例如：rgb 2*rgb。
