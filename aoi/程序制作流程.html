程序制作流程

程序制作流程

新建检测程序

  1. 打开Neo.AOI.exe主程序。
  2. 依次点击菜单项 [ AOI ] → [ 新建 ] ，新建空白检测程序。
  3. 修改参数机种名称值为实际机种名。

采集图像

  1. 设置检测程序参数PCB长度 PCB宽度 为电路板的实际长宽（单位：毫米）。
  2. 在AOI机器上放置好需要检测的电路板。
  3. 点击 [ 主页 ] → [ 采集图像 ] 开始采集电路图像。如果使用的是离线机型，在点击 [ 采集图像 ] 后还需要按机器上的绿色按钮。
  4. 等待机器扫描完电路板后，就可以看到采集回来的电路板图像。

导入CAD坐标数据

具体步骤请参考：导入CAD

制作Mark检测库

MARK检测库制作是检测程序制作过程中最重要的环节，只有找到MARK，才能校正图像和元件位置，后续各类元件检测库制作都将依赖于MARK的确定。具体操作步骤请参考：制作mark检测库。

制作其他元件检测库
