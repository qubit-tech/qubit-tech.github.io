AOI主菜单

AOI主菜单

菜单路径 菜单说明
[ AOI ] → [ 新建 ] 新建一个程序
[ AOI ] → [ 打开 ] → [ 程序 ] 打开后缀为.naoi的程序文件
[ AOI ] → [ 打开 ] → [ PCB图像 ] 打开后缀为.nimg的图像文件
[ AOI ] → [ 保存 ] 保存当前程序
[ AOI ] → [ 另存为 ] 当前程序另外保存一份
[ AOI ] → [ 导出 ] → [ 导出单面CAD ] 将电路板当前面别的CAD信息导出至指定位置
[ AOI ] → [ 导出 ] → [ 导出双面CAD ] 将电路板双面CAD信息导出至指定位置
[ AOI ] → [ 导出 ] → [ 导出图像 ] 将当前整板图无损保存至指定位置
[ AOI ] → [ 导出 ] → [ 导出图像有损压缩 ] 将当前整板图压缩后保存至指定位置
[ AOI ] → [ 导出 ] → [ 导出通用图像 ] 将当前整板图转化为普通电脑可以解析的图像（默认.jpg，100u）并保存在指定位置
[ AOI ] → [ 导出 ] → [ 导出检测库 ] 将当前程序所包含的所以检测库导出至指定位置
[ AOI ] → [ 导出 ] → [ 导出覆盖率 ] 将当前程序覆盖率导出成csv文件至指定位置
[ AOI ] → [ 导入 ] → [ 导入单面CAD ] 将选中CAD导入至程式当前面别，并覆盖掉原CAD信息
[ AOI ] → [ 导入 ] → [ 导入双面CAD ] 将选中双面CAD导入至当前程序，并覆盖掉原CAD信息
[ AOI ] → [ 导入 ] → [ 导入关联检测库 ] 将选中检测库导入当前程序，通过分析料号（组名称）信息，自动完成检测库链接，并清理掉未链接检测库
[ AOI ] → [ 导入 ] → [ 导入检测库 ] 将选中检测库导入当前程序，并通过分析料号（组名称）信息，自动完成检测库链接
[ AOI ] → [ 导入 ] → [ 导入检测程序 ] 将选中程序中所有检测库导入当前程序，并通过分析料号（组名称）信息，自动完成检测库链接
[ AOI ] → [ 登录 ] 人员登录
[ AOI ] → [ 修改密码 ] 修改密码
[ AOI ] → [ 注销 ] 注销当前账号登录
